＜概要＞  
画像ロールオーバー用プラグインです。  
ロールオーバー用の画像はプリロードします。  
  
＜セッティング＞  
jQueryを読み込む。  
jquery.slideImage.jsを読み込む。  
  
＜使い方＞  
（１）
&lt;img class="rollover" src="test.jpg"&gt;  
とすると、test_on.jpg をプリロード ＋ ロールオーバー時に画像を差し替え  
  
（２） 
&lt;img over="ロールオーバー用画像" src="アウト時の画像"&gt;
